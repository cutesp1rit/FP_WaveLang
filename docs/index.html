Добро пожаловать в Wave!

Wave — это функциональный язык программирования, вдохновленный идеей "волновых" преобразований данных. Язык построен на простых, но мощных концепциях: композиции функций, паттерн-матчинге и автоматическом применении функций к спискам.

Переходите в визуальный отладчик и тестируйте!

Композиция функций

Оператор ~ позволяет естественно соединять функции в цепочки преобразований.

Паттерн-матчинг

Определяйте функции с несколькими вариантами, проверяя условия элегантно и выразительно.

Автоматический map

Применяйте функции к спискам автоматически — язык сам понимает ваши намерения.

Рекурсия

Полная поддержка рекурсивных функций как основного механизма итерации.

Простой синтаксис

Минималистичный и понятный синтаксис, сосредоточенный на главном — ваших идеях.

REPL

Интерактивная среда для экспериментов и быстрого прототипирования.

Быстрый старт

Установка:

git clone https://github.com/yourusername/FP_WaveLang.git cd FP_WaveLang dotnet build

Запуск программы:

dotnet run --project src/WaveLanguage/WaveLanguage.fsproj examples/factorial.wave

Интерактивный REPL:

dotnet run --project src/WaveLanguage/WaveLanguage.fsproj

Пример кода

Вычисление факториала в Wave:

factorial ~ n n = 0 ~ 1 _ ~ n * factorial(n - 1) factorial(5) // Результат: 120

Композиция функций:

double ~ x ~ x * 2 addOne ~ x ~ x + 1 square ~ x ~ x * x 3 ~ double ~ addOne ~ square // Результат: 49

Работа со списками:

[1, 2, 3, 4, 5] ~ double // [2, 4, 6, 8, 10] [1, 2, 3, 4, 5] ~ square // [1, 4, 9, 16, 25]
